# Changelog

## v0.2.1 (2025-05-13)

### 新增功能

#### 国际化 (i18n) 支持
- 新增中英文切换功能，应用内所有按钮、说明文字支持中文/英文
- 统计面板右上角新增语言切换按钮（中文/EN）
- 系统托盘右键菜单支持中英文切换
- 托盘菜单新增"语言切换"选项
- 语言设置自动保存，重启后保持上次选择
- 100 种作物名称全部提供中英文翻译
- 默认语言设为中文

#### 平面视图模式
- 新增"平面"视图模式，键盘以水平 2D 方式显示
- 平面视图具有立体键帽效果（顶面、侧面、底部阴影三层结构）
- 按键按下时有真实的"下压"动画效果
- 平面视图沿 X 轴微倾 15°，增加透视感
- 支持作物 emoji、害虫、金色发光等所有游戏元素

#### 自由旋转
- 窗口底部中间新增旋转手柄控件
- 按住手柄上下左右拖动可自由调整 3D 视角
- 支持 X 轴 ±70° 和 Y 轴 ±60° 范围旋转
- 双击手柄重置回默认视角
- 手柄实时显示当前旋转方向指示器

#### 视图/视角切换
- 三种视图模式循环切换：农场 → 平面 → 热力图
- 两种视角切换：左 / 右
- 托盘菜单中显示当前选中项标识（✓）

#### 设置面板
- 托盘右键菜单新增"设置"选项，点击打开设置弹框
- 设置弹框支持视图模式切换（农场/平面/热力图）
- 设置弹框支持视角切换（左/右）
- 新增快捷键设置功能：
- **显示/隐藏窗口**：全局快捷键，默认为空，用户自定义设置后注册为系统级全局快捷键，任何时候都可触发
- **打开统计**：应用内快捷键，仅在应用窗口获得焦点时响应
- **锁定/解锁窗口**：应用内快捷键，锁定后窗口置于最高层且无法移动/调整大小
- 快捷键录入支持：点击输入框后按下组合键即可录入，支持清除
- 快捷键设置自动保存，重启后自动加载并注册

#### 窗口锁定
- 新增窗口锁定功能，锁定后窗口无法拖动和调整大小
- 锁定时窗口左上角显示 🔒 图标提示
- 支持通过自定义快捷键快速锁定/解锁
- 锁定状态下仍可正常操作农场（收获、除虫等）

### 改进

#### 托盘菜单汉化
- 所有托盘右键菜单项支持中英文
- 菜单项包括：显示/隐藏窗口、置顶/取消置顶、视图切换、视角切换、设置、统计、语言、退出

#### 窗口优化
- 隐藏任务栏图标（`skipTaskbar: true`），仅保留托盘图标
- 移除窗口阴影（`shadow: false`），消除透明窗口四周的"盒子"效果
- 修复 Windows 上拖动窗口时的抖动/抽搐问题（macOS resize workaround 限制为仅 macOS 执行）
- 拖动边框仅在鼠标按下时显示，不再因 resize 事件误触发

### 文档
- README.md 全部翻译为中文
- 新增语言切换功能说明
- 新增 CHANGELOG.md

### 技术细节
- 新增 `src/i18n/` 目录：类型定义、中英文翻译文件、Context Provider
- 前端使用 React Context + Tauri Store 持久化语言偏好
- 后端通过 `emit("set-language")` 事件与前端同步语言状态
- Rust 端读写 `store.json` 持久化语言设置
- FarmCanvas 新增 `'flat'` viewMode 渲染分支
- CSS 3D transform (`perspective` + `rotateX` + `rotateY`) 实现自由旋转
- 新增 `SettingsPanel` 组件，支持视图/视角/快捷键配置
- Rust 端新增 `set_toggle_shortcut` 命令，支持动态注册/注销全局快捷键
- 全局快捷键通过 `tauri_plugin_global_shortcut` 的 `register` + `with_handler` 统一处理
- 应用内快捷键通过前端 `keydown` 事件监听实现
